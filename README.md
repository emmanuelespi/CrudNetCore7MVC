# Información general
Este es un proyecto para entender el funcionamiento de un CRUD hecho en net Core 7 con ASP .net y con servidor de SQL SERVER.

# Tecnologías usadas

1. Visual Studio 2022
2. .net Core 7
3. C# 
4. Páginas Blazor
